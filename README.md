# Web Total Solution - Landing Page

## Quick Start

1. **Extract the archive:**
   ```bash
   tar -xzf web-total-solution-complete.tar.gz
   cd web-total-solution-complete/
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   - Navigate to `http://localhost:5000`

## Dependencies Required

Make sure you have Node.js 18+ installed. The project includes:

- Next.js 14 (App Router)
- React 18
- TypeScript
- TailwindCSS
- Framer Motion
- GSAP with ScrollTrigger
- Lenis (smooth scrolling)

## Build for Production

```bash
npm run build
npm start
```

## Features

- ✨ Custom smoky cursor with particle effects
- 🎯 Smooth scroll animations (Lenis + GSAP)
- 📱 Fully responsive design
- ♿ Accessibility compliant
- ⚡ Performance optimized
- 🚀 Ready for deployment

## Troubleshooting

If you encounter issues:

1. Make sure you're using Node.js 18+
2. Delete `node_modules` and run `npm install` again
3. Clear Next.js cache: `rm -rf .next`
4. Restart the development server
